# url router

#### 产品列表

```
/product/list?category=null&span_city=null&start_city=null&product_type=null&item_type=null
参数：
category=null&
span_city=null&
start_city=null&
product_type=null&
item_type=null
```

#### 产品详情

```
/product/detail?product_id=1398
参数：
product_id=1398
```
